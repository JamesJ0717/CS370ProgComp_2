# CS370ProgComp_2

CS370 Programming Competition Group 2

## Installation

-   Download the files in the repo

    -   Either clone the repo
        `git clone https://github.com/jamesj0717/CS370ProgComp_2`
    -   Or download a zip

    -   Install the dependencies  
        `body-parser`  
        `express`  
        `formidable`  
        `fs`  
        `http`  
        `mongoose`  
        `morgan`  
        `multer`

-   Run  
    `node server` to start the server
